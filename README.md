# 🍽️ Sabor & Estilo - Landing Page de Restaurante

Este proyecto es una **Landing Page estática e informativa** diseñada para un restaurante gourmet. Es el primer entregable del programa, enfocado en el dominio de las tecnologías base del Front-End: HTML5, CSS3, Bootstrap 5 y manipulación del DOM con JavaScript.

## 🎯 Objetivo del Proyecto
Desarrollar una interfaz profesional, responsiva y funcional que resuelva problemas de visibilidad y gestión básica de reservas para el cliente, siguiendo estándares de código limpio y semántico.

---

## 🚀 Tecnologías y Herramientas
* **HTML5**: Estructura semántica completa utilizando etiquetas como `<main>`, `<section>` y `<article>`.
* **Bootstrap 5**: Implementación del **Grid System** para lograr un diseño 100% responsive y uso de componentes como Navbar, Cards y Forms.
* **CSS3**: Estilos personalizados, efectos de hover, tipografías de Google Fonts y diseño de sección Hero con imagen de fondo.
* **JavaScript (Vanilla)**: Lógica centrada en la manipulación fundamental del DOM, captura de eventos y validación de datos.

---

## 🛠️ Funcionalidades Destacadas
1. **Diseño Responsivo**: Adaptación fluida para dispositivos móviles, tablets y escritorio mediante el sistema de columnas de Bootstrap.
2. **Navbar Dinámica**: Menú colapsable con comportamiento inteligente que se cierra automáticamente al seleccionar una sección en móviles.
3. **Galería de Menú**: Visualización de 6 platos principales utilizando un sistema de tarjetas (cards) bien distribuidas.
4. **Sistema de Reservas**: Formulario funcional que captura los datos del usuario, muestra alertas de confirmación y actualiza el contenido del DOM sin recargar la página.
5. **Footer Profesional**: Diseño con alto contraste y enlaces directos a redes sociales simuladas.
